# Counter-Evidence

Use this file when the task is high-stakes, contentious, fast-moving, commercial, comparative, predictive, or likely to be distorted by hype, marketing, or selective evidence.

Do not stop at the first coherent explanation. Actively look for evidence that could weaken, limit, or overturn your current conclusion.

## Core rule

For every high-impact conclusion, ask:

- What evidence would weaken this conclusion?
- What evidence would directly contradict it?
- What evidence would show it only holds in a narrow context?
- What evidence would show the apparent signal is misleading?

## When counter-evidence is mandatory

Always run a counter-evidence pass when the research involves:

- company strength or startup quality
- market opportunity or category growth
- product superiority or competitive claims
- technical feasibility or performance claims
- vendor selection
- buy/build/partner decisions
- practical regulatory interpretation
- predictions or recommendations

## Common counter-evidence angles

Check for:

1. direct contradiction
2. scope limitation
3. time decay
4. incentive distortion
5. hidden trade-offs
6. failure cases
7. alternative explanations

## Search patterns

Use searches that try to break the thesis, for example:

- `criticisms of [topic]`
- `problems with [topic]`
- `limitations of [topic]`
- `[topic] complaints`
- `why [topic] failed`
- `[topic] controversy`
- `[topic] alternatives`
- `[topic] hidden costs`
- `[topic] security issues`

## How to run the pass

1. Write the current working conclusion in one sentence.
2. List the biggest assumptions behind it.
3. Search for evidence that weakens those assumptions.
4. Find at least one independent contradiction or limiting case.
5. Reassess confidence.
6. Update the final answer.

## Handling conflicting evidence

When support and counter-evidence conflict, compare:

- source quality
- independence
- recency
- directness
- methodological clarity
- relevance to the exact claim

Prefer:

- primary over secondary
- measured evidence over opinion
- recent evidence over stale evidence for fast-moving topics
- independent reporting over repeated citation chains

If conflict cannot be resolved, lower confidence and say so clearly.

## Output expectations

Include some version of:

- strongest supporting evidence
- strongest counter-evidence
- what remains unresolved
- how counter-evidence affected confidence
- what would change the conclusion
