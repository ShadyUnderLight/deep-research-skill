# First-Tier Positioning Memo Example

## Example task

Assess whether an entity can reasonably be treated as top-tier or first-tier without collapsing multiple dimensions too early.

## Expected route

Primary route:

- first-tier / competitive positioning

## Expected secondary disciplines

Typical supporting disciplines include:

- source traceability
- evidence weighting
- counter-evidence
- current-state verification when the task is time-sensitive

## Expected visible artifact contract

A strong first-tier positioning memo should make the following visible:

- scope
- metric
- timeframe
- dimension-level conclusions
- overall-label gate
- direct evidence vs inference-heavy judgment
- what weakens the label

## Typical failure signs

Common failures include:

- jumping to a prestige label too early
- doing dimension-level work but collapsing it into one label without a gate
- making evidence weighting invisible
- using confirmed / inference / uncertainty language without letting it affect the judgment
- presenting an overall label without a visible threshold for using it

## Minimal output shape

A compact first-tier positioning memo often includes:

- executive answer
- scope / metric / timeframe
- dimension-by-dimension conclusions
- overall-label gate
- what weakens the label
- open uncertainty
- sources
