# Freshness Failure Cases

Use these as test prompts when checking whether the skill verifies the current state before analysis.

## Example prompts

1. Research the current Xiaomi flagship phone lineup and explain how it compares with Samsung.
2. Compare the latest GPT and Claude model families for coding use right now.
3. Analyze the current pricing and plan differences for Notion, ClickUp, and Asana.
4. Research the current leadership and strategic direction of OpenAI.

## What a good run should do

- verify the current snapshot first
- prefer official current pages over old summaries
- make dates visible where relevant
- avoid mixing historical and current states
- lower confidence if current state cannot be verified
