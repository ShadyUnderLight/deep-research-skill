# Eval: Unitree Competitive Positioning Secondary Route + Label Case

## Goal

Test whether a Competitive Positioning / First-tier report with strong structural execution (dimension-level conclusions, evidence-strength separation, explicit overall-label gate) can still achieve only Conditional Pass when secondary route hard-fails are not itemized, secondary/media sources support strong positioning claims without sufficient source role labeling, and self-assessment overclaims pass status.

## Real case pattern

A user-provided 宇树科技 (Unitree Robotics) global humanoid robot first-tier positioning report dated **2026-06-05** demonstrates this pattern.

**What was done well:**
- ✅ Competitive Positioning route correctly selected with scope/metric/timeframe
- ✅ 4-dimension analysis (hardware, AI, commercialization, capital markets) with per-dimension conclusions
- ✅ Weighted aggregation rules with explicit overall-label gate → Conditional/Weak Tier 1
- ✅ Body-level `[Sxx]` citations present throughout
- ✅ Structured source register (§255-276) with ID, type, date, URL
- ✅ Strong counter-evidence (AI gap, growth slowdown, competition, geopolitical, valuation risk)
- ✅ Uncertainty register with verification paths (§207-215)
- ✅ Judgment-first opening with conditional framing
- ✅ Market position claims mostly scoped with dimension/geography

**Core issues (conditional pass level):**
- ❌ **Secondary route hard-fail not itemized** — declares startup-evaluation + technical-deep-dive as secondary but startup checklist was skipped with no per-condition verification
- ❌ **Source-traceability self-assessment overconfident** — claims ✅ passed but S01 (core source) has no URL; secondary media/blog/aggregator sources support strong competitive claims without sufficient source role downgrade
- ❌ **Strong claims need stronger verification** — "全球唯一盈利", "性价比无对手", "市场最低价" carried by secondary/analyst sources without exclusivity verification or explicit downgrade
- ❌ **Quantitative role labels incomplete** — important numbers lack observed/estimate/vendor-claim/analyst-inference markers
- ❌ **Valuation cited without method** — IPO valuation and 25x PS used for risk but no comparable multiples, scenario framework, or methodology

## Scoring

- **Full Pass**: secondary hard-fails itemized + honest source labels + verified strong claims
- **Conditional Pass**: strong primary route execution but secondary + label gaps (this case's level)
- **Fail**: primary route hard-fail triggered
