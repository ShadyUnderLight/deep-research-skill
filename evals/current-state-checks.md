# Current-State Checks

Lightweight evaluation prompts for freshness and current-state verification.

## Pass criteria

A good answer should:

- verify current-state facts before broader analysis
- cite or clearly rely on current primary sources
- avoid stale product/version mixups
- distinguish verified current facts from historical context
- state uncertainty when current verification is incomplete

## Suggested prompts

- What is Xiaomi's current flagship lineup, and how should a buyer think about it?
- Compare the current public pricing of three major project-management tools.
- Research the latest major Claude and GPT coding models and summarize practical differences.
