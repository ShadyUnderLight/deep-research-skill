# Eval: Research Depth Rubric

Use this rubric after a real report run when you want to judge research depth and breadth rather than only factual correctness.

## Goal

Measure whether a report behaves like real deep research instead of a well-written summary.

## Dimensions

Score each from **0-2**.

### 1. Question framing
- 0 = did not clearly identify the real objective
- 1 = some framing, but broad or generic
- 2 = clearly framed the true objective and useful subquestions

### 2. Coverage breadth
- 0 = major relevant dimensions missing
- 1 = key dimensions partly covered
- 2 = important dimensions covered with good prioritization

### 3. Evidence depth
- 0 = weak sourcing or little primary/independent support
- 1 = mixed evidence quality
- 2 = strong, layered, and relevant evidence for key claims

### 4. Analytical depth
- 0 = mostly summary
- 1 = some analysis, but shallow or uneven
- 2 = clear reasoning, trade-offs, and implications

### 5. Counter-evidence depth
- 0 = little meaningful challenge to the thesis
- 1 = some counter-evidence, but weak or token
- 2 = counter-evidence materially improves calibration

### 6. Decision usefulness
- 0 = informative but not very actionable
- 1 = some bottom line, but weak next-step value
- 2 = clearly useful for decision-making

## Total score guide

- 0-4 = shallow
- 5-8 = moderate
- 9-12 = strong deep-research quality

Use this rubric on:

- improved-but-still-flawed reports
- reports that are factually decent but feel shallow
- reports where you want to compare old vs new skill behavior
- reports that feel impressive but may be mostly synthesis theater
