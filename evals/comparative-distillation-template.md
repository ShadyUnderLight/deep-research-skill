## Minimal quality bar

Before finalizing this file, check:

- [ ] the two reports are comparable enough to justify distillation
- [ ] the comparison used the six fixed dimensions
- [ ] each accepted candidate has an action type
- [ ] each accepted candidate has a proposed repo home
- [ ] at least one rejected observation is documented when relevant
- [ ] the final judgment distinguishes rule gap vs trigger gap vs execution gap

If these boxes are not checked, the file is probably commentary rather than comparative distillation.
