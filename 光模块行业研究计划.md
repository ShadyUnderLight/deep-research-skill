# 光模块行业研究计划

## 研究目标
分析光模块行业竞争格局，重点关注中际旭创 (300308.SZ) 的行业位置。

## 路由选择
**主要路线：竞争定位**
**最接近的替代路线：市场展望**

## 路由执行契约
- **主要路线：** 竞争定位
- **最接近的替代路线：** 市场展望
- **开场必须做：** 明确中际旭创在全球光模块行业中的竞争地位，维度分析
- **必需章节：** 竞争格局分析、中际旭创竞争优势与劣势、技术趋势、下游需求
- **最小化/延后：** 详细的公司财务分析（可以简要提及）
- **路由激发的可见证明：** 维度级别结论、证据权重分离、显式整体标签门
- **需要警惕的硬失败迹象：** 使用“第一梯队”等词汇过于宽松，将多个维度合并为声望标签

## 核心子问题
1. 全球光模块市场规模与增长趋势
2. 主要竞争对手分析（新易盛、光迅科技、华工科技、Lumentum、II-VI等）
3. 中际旭创的市场份额与竞争地位
4. 行业技术趋势（800G、1.6T、CPO等）
5. 下游需求驱动（AI、数据中心、5G等）

## 证据标准
- 优先使用行业报告、券商研报、权威媒体
- 明确区分：确认事实、推断、未知
- 为每个重要结论提供来源URL

## 已收集数据摘要

### 市场规模数据
- 2024年全球光模块市场规模：146亿美元（Lightcounting数据）或178亿美元（其他来源）
- 2025年预计：230-238亿美元（LightCounting预测），增长50%
- 2029年预计：415亿美元（CAGR 18.5%）
- 中国市场：2024年606亿元人民币，2025年预计700亿元人民币，2029年872亿元人民币（CAGR 21.5%）

### 中际旭创数据
- 2024年全球市场份额约23%，全球第一
- 2024年光通信收发模块业务收入228.86亿元，同比增长124.77%
- 2024年营业收入238.62亿元，同比增长122.64%
- 2024年净利润51.71亿元，同比增长137.93%
- 800G光模块市场份额超过40%
- 2023年首次成为全球第一大光模块厂商

### 竞争格局
- 2023年全球前十大光模块厂商中，中国占7席，市场份额超过50%
- 第一梯队：光模块业务收入大于200亿元（中际旭创）
- 第二梯队：80-100亿元（新易盛、中天科技、光迅科技等）
- 第三梯队：10-20亿元以下（长飞光纤、太辰光等）
- 全球CR5超过50%，中国企业占据主导地位

### 技术趋势
- 800G成为主流，2024年规模商用
- 1.6T光模块2025年开始小批量出货，预计2026年成为主流
- CPO（共封装光学）技术，预计2026年后逐步商用
- 硅光技术凭借高集成度、低功耗优势，在800G及以上市场渗透率快速提升

### 下游需求驱动
- AI算力需求爆发，数据中心光模块需求激增
- 北美四大云厂商2025年资本开支总额超3300亿美元，同比增幅超50%
- 5G网络建设稳步进行，推动电信光模块需求
- 东数西算、算力网络等政策推动

## 停止条件
- 主要问题得到充分回答
- 新搜索 mostly repeat prior findings
- 剩余不确定性是由于数据不可用，而非缺乏努力

## 输出格式
结构化的研究发现，包括：
1. 行业概况（市场规模、增长率、主要玩家）
2. 竞争格局（市场份额、竞争梯队）
3. 中际旭创的竞争优势与劣势
4. 行业技术趋势与路线图
5. 下游需求分析
6. 信息来源列表
7. 不确定性说明
